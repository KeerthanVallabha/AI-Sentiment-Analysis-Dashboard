# AI Sentiment Analysis Dashboard

Social Media Sentiment Analysis Dashboard built using Python, NLP and Streamlit.

## Features

- Sentiment prediction
- Interactive dashboard
- Pie charts
- Bar graphs
- Word cloud
- Dataset upload
- Download analyzed results

## Technologies Used

- Python
- Streamlit
- TextBlob / NLP
- Pandas
- Plotly
- Matplotlib

## Run Project

```bash
python -m streamlit run app.py
```

## Project Output

Analyzes social media posts and predicts sentiments:
- Positive
- Negative
- Neutral
